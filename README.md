#  Image-Editor

## Overview

**Image-Editor** is a browser-based application that allows users to upload and edit images directly within their web browser. Built using core web technologies, the project provides real-time visual feedback, making image manipulation simple, interactive, and efficient.

## Features

* Upload images directly from your device
* Real-time preview of edits
* Image adjustments (brightness, contrast, filters)
* Crop, resize, and rotate functionality
* Undo and redo support for editing actions
* Download/export edited images
* User-friendly and interactive interface
* Fully browser-based (no installation required)

## Tech Stack

* **HTML5** – for structuring the application
* **CSS3** – for styling and layout
* **JavaScript** – for functionality and real-time image processing

## Getting Started

To run this project locally:

1. Clone the repository

   ```
   git clone https://github.com/your-username/Image-Editor.git
   ```
2. Navigate to the project folder

   ```
   cd Image-Editor
   ```
3. Open `index.html` in your browser
